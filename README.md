# @neokekeke/k-ui 组件库

## 安装

```
npm i @neokekeke/k-ui
```

## 用法
```
import Vue from 'vue'
import NeoUI from '@neokekeke/k-ui'
import '@neokekeke/k-ui/lib/css/k-ui.css';

Vue.use(NeoUI)

<!-- 引入Button组件 -->
<template>
  <div class="test-container">
   <n-button></n-button>
  </div>
</template>
```
